1. Simplify
(a) $3 a^{2} b \times\left(2 a b^{2}\right)^{2}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
3 a^{2} b \times\left(2 a b^{2}\right)^{2} &=3 a^{2} b \times\left(4 a^{2} b^{4}\right)\\
&=12 a^{4} b^{5}
\end{align*}

</div>

+++

<br>

(b) $10^{-1 / 3} \times 25^{2 / 3} \div 2^{5 / 3}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

Change the base to 2 and 5 so that we can combine the terms more easily.

\begin{align*}
10^{-1 / 3} \times 25^{2 / 3} \div 2^{5 / 3} &=\frac{25^{2 / 3}}{10^{1 / 3} \times 2^{5 / 3}}\\
&=\frac{(5^{2})^{2 / 3}}{(2 \times 5)^{1 / 3} \times 2^{5 / 3}}\\
&=\frac{5^{4 / 3}}{2^{1 / 3} \times 5^{1 / 3} \times 2^{5 / 3}}\\
&=\frac{5^{(4 / 3)-(1 / 3)}}{2^{(1 / 3)+(5 / 3)}}\\
&=\frac{5^{3 / 3}}{2^{6 / 3}}\\
&=\frac{5}{2^{2}}\\
&=\frac{5}{4}
\end{align*}

</div>

+++

<br>

\((c)\) $\frac{\left(a^{3}\right)^{5}}{\left(a^{2}\right)^{5 / 2}}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
\frac{\left(a^{3}\right)^{5}}{\left(a^{2}\right)^{5 / 2}} &=\frac{a^{15}}{a^{5}}\\
&=a^{15-5}\\
&=a^{10}
\end{align*}

</div>

+++

<br>

(d) $32^{3 / 2} \times 8^{5} \times 2^{-5 / 2}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">


\begin{align*}
32^{3 / 2} \times 8^{5} \times 2^{-5 / 2} &=\left(2^{5}\right)^{3 / 2} \times\left(2^{3}\right)^{5} \times 2^{-5 / 2}\\
&=2^{(5 \times 3 / 2)} \times 2^{(3 \times 5)} \times 2^{-5 / 2}\\
&=2^{15 / 2} \times 2^{15} \times 2^{-5 / 2}\\
&=2^{15 / 2} \times 2^{30 / 2} \times 2^{-5 / 2}\\
&=2^{(15 / 2)+(30 / 2)+(-5 / 2)}\\
&=2^{40 / 2}\\
&=2^{20}
\end{align*}

</div>

3. Simplify as far as possible:
(a) $\sqrt{96}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
\sqrt{96} &=\sqrt{16 \times 6}\\
&=4 \sqrt{6}
\end{align*}

</div>

+++

<br>

(b) $\frac{2-\sqrt{3}}{\sqrt{3}}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
\frac{2-\sqrt{3}}{\sqrt{3}} &=\frac{(2-\sqrt{3}) \times \sqrt{3}}{\sqrt{3} \times \sqrt{3}}\\
&=\frac{2 \sqrt{3}-3}{3}
\end{align*}

</div>

+++

<br>

\((c)\) $\frac{1+\sqrt{2}}{3-\sqrt{2}}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
\frac{1+\sqrt{2}}{3-\sqrt{2}} &=\frac{(1+\sqrt{2}) \times(3+\sqrt{2})}{(3-\sqrt{2}) \times(3+\sqrt{2})}\\
&=\frac{3+4 \sqrt{2}+2}{9-2}\\  
&=\frac{5+4 \sqrt{2}}{7}
\end{align*}

</div>

+++

<br>

(d) $\sqrt{98}-\sqrt{32}$

+++ <span style="color:green">Solutions</span>

<div style="font-size: 22px">

\begin{align*}
\sqrt{98}-\sqrt{32} &=\sqrt{49 \times 2}-\sqrt{16 \times 2}\\
&=7 \sqrt{2}-4 \sqrt{2}\\   
&=3 \sqrt{2}
\end{align*}

</div>
